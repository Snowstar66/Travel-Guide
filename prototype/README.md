# Trip Companion NYC Prototype

## Quick Preview

Open `index.html` directly in a browser, or run a tiny local server:

```powershell
python -m http.server 4173 -d prototype
```

Then open:

`http://localhost:4173`

## Included in This Prototype

- 5-day NYC first-timer guide
- day-by-day detail view
- local check-off state
- favorites
- notes saved in browser local storage
- know-before-you-go cards
